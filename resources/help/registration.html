Регистрация пользователей через файл

Для регистрации пользователей через CSV-таблицу обязательно нужно определить следующие колонки (первая строка отводится под заголовки):

  1. lastname - фамилия регистрируемого пользователя;
  2. firstname - имя регистрируемого пользователя;
  3. recoveryEmail - запасной адрес электронной почты (не из основного домена). На него придет письмо с автоматической рассылкой;
  4. primaryEmail - новый адрес электронной почты на основном домене (например, "example.com"). Если вы не хотите создавать адреса вручную, то можете у первого регистрируемого пользователя указать основной домен. Программа автоматически создаст всем последующим пользователям адрес электронной почты, который будет выглядеть как lastname.firstname@example.com. Вся кириллица будет представлена в траслитерации;
  5. orgUnitPath - указываете иерархию организационного подразделения с корня. Пример: /Работники/Бухгалтерия/Расчетный отдел.

Дополнительные поля для регистрации пользователей:

  1. recoveryPhone - номер мобильного телефона. Обязательно в формате: +7 (000) 000 00-00;
  2. employeeId - уникальный идентификатор работника. Вы можете ввести значения вручную (из ваших систем аутентификации), либо же вы можете сгенерировать номер автоматически, указав у первого регистрируемого пользователя слово auto;
  3. workAddress - рабочий адрес работника. Если адресов несколько, то адреса указывайте в двойных кавычках, разделенными запятыми;
  4. homeAddress - домашний адрес работника. Если адресов несколько, то адреса указывайте в двойных кавычках, разделенными запятыми;
  5. changePassword - укажите FALSE, если вы хотите, чтобы при следующем входе система запросила смену пароля. В противном случае, ничего не указывайте, либо пишите TRUE;
  6. employeeStatus - статус работника. Может быть Active (Активный), Suspended (Заблокированный) и Achived (Архивированный). По умолчанию - Active

Пример CSV-таблицы:

lastname firstname recoveryEmail primaryEmail password orgUnitPath recoveryPhone employeeId workAddress homeAddress changePassword employeeStatus
